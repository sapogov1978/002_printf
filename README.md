## Usage

### Requirements

The project is written in C language and thus needs the **`gcc` compiler** and some standard **C libraries** to run.

### Instructions

**1. Compiling**

To compile the library, run:

```shell
$ cd [path/to/project] && make
```

**2. Running**

To use the functions:

```shell
./gnl test_cases/[any_case]
```
